Homework 5: Due Friday 4 February at the beginning of class As on the previous assignment, you can work in pairs and hand in only one set of answers if you want.

  • Logistic Explorations. Consider the following parameter values:
    1. 3.594
    2. 3.835
    3. 3.567
    4. 3.7
    5. 3.5699456718695445
    For each of the parameter values, describe the behavior of iterates of the logistic map. Specifically, you should answer the following questions for each r value:
    • Is the map periodic or chaotic or something else? To answer this question you'll probably want to use the orbit program, available here.
    • If the system is periodic, what is the period?
    • Does the system possess sensitive dependence on initial conditions? To answer this questions, you'll need the program I showed you briefly in class, available here.
    For the last one, resist the temptation to round off the r value. Do not be surprised if you get seemingly contradictory results.
