Handouts from Class



  • Chapter Four: Graphics
    1. graphics_example_2.py. Basics of Zelle's graphics package.
    2. graphics_example_4.py. Illustrates how to get text from an entry box and open multiple windows.
    3. graphics_example_5.py. Illustrates how use the setCoords function in Zelle's graphics package.
    4. graphics_example_6.py. Illustrates different ways to set colors in Zelle's package. Also shows how to use the randrange function to set random colors. package.
  • Chapter Five: Text Processing
    1. file_output_example.py. A simple example writing data to a file.
    2. text2numbers.py. Convert text to numbers.
    3. numbers2text.py. Convert numbers to text.
    4. file_input_example.py. Example showing how to input data from a file.
    5. file_input_example.2.py. Another example showing how to input data from a file.
  • Chapter Six: Functions
    1. dave_function_1.py. A simple function example.
    2. dave_function_5.py. Overview of functions.
  • Chapter Seven: Control Statements
    1. dave_if_1.py. Simple if statements.
    2. dave_max.py. Different ways of illustrating a max function.
  • Chapter Eight: More Control Statements
    1. dave_for_review.py. Review of for loops.
    2. dave_while_1.py. Intro to while statement
    3. dave_while_2.py. More while loop, including a "sentinel" loop.
    4. dave_while_3.py. An end of file loop.
    5. dave_boolean_1.py. An introduction to Boolean statements.
    6. dave_boolean_2.py. Using expressions as Boolean statements.
    7. dave_break.py. Using a break statement to exit a while loop.
  • Chapter 10: Classes
    1. dave_class_review.py. A short program that reviews classes.
    2. dave_class_student.py. A program that illustrates how to define classes.
    3. dave_class_purple_sheep.py. A program involving purple sheep. Not sure where this is headed.
